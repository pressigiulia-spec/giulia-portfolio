GIULIA PRESSI - PORTFOLIO NETLIFY

Questa cartella contiene il sito completo pronto per Netlify.

Struttura principale:
- index.html
- css/style.css
- js/main.js
- projects/
- assets/images/
- assets/videos/
- assets/tshirt/
- assets/originals/

Per pubblicarlo su Netlify:
1. Carica l'intera cartella giulia-portfolio-netlify.
2. Usa index.html come homepage.
3. Non servono build command o framework.
